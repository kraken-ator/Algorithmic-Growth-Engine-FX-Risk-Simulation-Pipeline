# Algorithmic Growth Engine & FX Risk Simulation Pipeline 🚀
*A Quantitative Go-To-Market (GTM) strategy model engineered for Fin-Tech B2B cross-border payments.*

### 📊 Business Objective
Designed to optimize Customer Acquisition Cost (CAC) against Lifetime Value (LTV) by algorithmically identifying high-value MSME target segments.

### ⚙️ Technical Architecture
1. **Synthetic Market Generation:** Python script simulating 5,000+ MSME profiles across multiple industry niches and payment gateways.
2. **Stochastic FX Risk Modeling:** Implemented Monte Carlo simulations to calculate Value at Risk (VaR) from USD/INR volatility, proving the financial utility of flat-fee pricing models.
3. **Machine Learning Propensity Model:** Trained a Random Forest classifier (Scikit-Learn) to predict customer conversion probability based on historical fee losses and transaction volume.
4. **Dynamic CAC Optimization:** Algorithmic budget allocation determining the maximum viable marketing spend per cohort.
5. **Interactive Business Intelligence:** Power BI dashboard deployed to synthesize complex risk data into actionable GTM strategies via Quadrant Matrices and Dual-Axis ROI charts.

### 📂 Files Included
* `generator.py`: The core simulation and ML prediction engine.
* `advanced_growth_engine.csv`: The output dataset containing optimized CAC metrics.
* `dashboard.pbix`: The interactive Power BI visualization.
